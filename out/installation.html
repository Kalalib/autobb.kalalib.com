Local install package (Load unpacked)

Chrome Edge Developer Mode
  1. Download AutoBlackboardDownloader.zip.
  2. Extract the zip into a normal folder (do not keep it compressed).
  3. Open chrome://extensions or edge://extensions.
  4. Enable Developer Mode.
  5. Click Load unpacked and select the extracted folder.
  6. Pin AutoBB and open Blackboard to start downloading.

Before You Start

You cannot install directly from the zip file. Download and extract it first, then use Load unpacked in your browser extension page.

Install Time

Usually 1-2 minutes once the zip is downloaded.

Troubleshooting

Common fixes if installation does not work on first try.

Manifest error

Make sure you selected the extracted folder, not the zip file itself.

Load unpacked is disabled

Developer Mode is still off. Toggle it on and refresh the extensions page.

Extension not visible

Click the puzzle icon and pin AutoBB so it appears on your browser toolbar.
